Features

  • Drag and drop items to sort items
  • Hover over items to see the item's name and ID
  • Right click items to turn the background red
    • Good for tracking what you have
    • Items highlighted red will be removed if "Remove Selected" button is pressed
  • Import order from RSC+ generated CSV file
  • Export order to RSC+ compatible CSV file
  • Add and remove items from the bank grid
    • Can press enter in text field to add item

About this page

This webpage's source code is available at https://github.com/RSCPlus/rsc-bank-organizer.

It was originally created by The Legend of Brian in 2016 to plan how to organize his bank, but it has been expanded here to help sort everyone's bank, automatically.
